Automatically add a voice to your page content

== Description ==
Automatically add a voice to your page content thought <a href="https://responsivevoice.org/">responsivevoice API</a> and <a href="https://www.google.com/voice">Google Voice</a>.

You need to get a free **API key from responsivevoice** to make the plugin work : [Register](https://responsivevoice.org/register)

You can :

* Choose between 51 languages
* Add voice player to any custom post type or page
* Add voice player at the begining/ending of your page
* Change pitch, rate and volume
* Use a shortcode to add the player


You can use the shortcode <i>[mpv]</i>.
